    Get up and running with MetaMap in minutes

  <Steps>
    <Step title="Add the SDK to your project">
      Add the MetaMap SDK dependency to your app's `build.gradle` file:

      ```gradle
      dependencies {
          implementation 'com.metamap:android-sdk:3.42.17'
      }
      ```
    </Step>

    <Step title="Initialize the verification flow">
      Use `MetamapSdk.startFlow()` to launch a verification session with your client ID and flow ID:

      ```java
      MetamapSdk.INSTANCE.startFlow(
          this,
          "YOUR_CLIENT_ID",
          "YOUR_FLOW_ID",
          new Metadata.Builder().build()
      );
      ```
    </Step>

    <Step title="Handle the verification result">
      Capture the verification result using `onActivityResult` or the Activity Result API:

      ```java
      @Override
      protected void onActivityResult(int requestCode, int resultCode, Intent data) {
          if (requestCode == MetamapSdk.DEFAULT_REQUEST_CODE) {
              if (resultCode == RESULT_OK && data != null) {
                  String verificationId = data.getStringExtra(MetamapSdk.ARG_VERIFICATION_ID);
                  String identityId = data.getStringExtra(MetamapSdk.ARG_IDENTITY_ID);
                  // Handle success
              }
          }
      }
      ```
    </Step>
  </Steps>
